Hitman 2 Statistics
============

Hitman 2 real-time statistics tracker.

Features:<br>
 • Current mission name & number<br>
 • Timer with 3 decimals<br>
 • Shots fired<br>
 • Close encounters<br>
 • Headshots<br>
 • Alerts<br>
 • Enemies killed<br>
 • Enemies harmed<br>
 • Innocents killed<br>
 • Innocents harmed

How to use:<br>
  Run the program while playing the game, during a mission all values will be updated automatically.
